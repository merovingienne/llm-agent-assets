# CLI Agent Assets

A curated collection of agents, skills, and tools for LLM-powered CLI coding tools.

## Overview

This repository serves as a personal toolkit collecting various configurations, agents, and customisations designed to enhance productivity when working with AI-powered development tools.

## Supported Tools

There are separate collections of assets for different tools:

- **Claude Code** - Anthropic's official CLI for Claude
- **Google Gemini** - Google's AI coding assistant
- **OpenAI Codex** - OpenAI's code generation capabilities

## Contents

- **Agents** - Custom AI agents for specific tasks
- **Skills** - Reusable skill definitions and configurations
- **Tools** - Utility scripts and helper functions
- **Configurations** - Settings and preferences for various CLI tools

## Usage

Each asset in this collection includes its own usage instructions. Refer to the specific directory or file for detailed documentation on how to integrate it into your workflow.

## Contributing

This is a personal collection, but suggestions and improvements are welcome through issues and pull requests.
